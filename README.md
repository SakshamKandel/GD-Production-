# GD Production

Premium creative media & production company website built with React, Vite, Tailwind CSS, and GSAP.

## Tech Stack
- **React 18** — UI framework
- **Vite** — Build tool
- **Tailwind CSS** — Styling
- **GSAP** — Scroll-triggered animations
- **Lenis** — Smooth scrolling engine

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
